About This Project

This is a project using node js (without express js). This project exists because I was wondering how to make a simple crud app using only node js, so I started this project from an empty js file.
This project came from my friend who asked how to normalize the database from the sample below and this is my ERD.
The most difficult thing in this project is how to serve static files. Shadow DOM is also used to create reusable components.
